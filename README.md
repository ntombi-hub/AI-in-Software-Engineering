# AI in Software Engineering Assignment

## Overview
This project demonstrates AI applications in software engineering:
- AI code completion (GitHub Copilot)
- Automated testing (Selenium/Testim.io)
- Predictive analytics (Random Forest)
- Ethical reflections

## Folder Structure
- code_completion/ → Task 1
- automated_testing/ → Task 2
- predictive_analytics/ → Task 3
- bonus_innovation/ → Bonus Task
- report.pdf → Theoretical + screenshots
- requirements.txt → Dependencies

## How to Run
1. Install dependencies: pip install -r requirements.txt  
2. Run notebooks/scripts in VS Code or Google Colab
